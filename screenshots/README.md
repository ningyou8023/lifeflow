# App 截图文件夹 (Screenshots)

此文件夹用于存放 LifeFlow App 的演示截图，供 GitHub 主页 `README.md` 调用。

请将 App 运行时的截图复制到此文件夹，并按照以下文件名重命名，以便在主页正确显示：

*   `banner.png` - 项目横幅（可选）
*   `home.png` - 首页截图
*   `pantry.png` - 物品管理页面截图
*   `journal.png` - 情绪日记页面截图
*   `focus.png` - 专注白噪音页面截图
*   `wallet.png` - 隐私钱包页面截图
*   `subs.png` - 订阅管理页面截图

> **提示**：主页的 `README.md` 已经配置好引用这些文件名的代码，只要文件存在，图片就会自动显示。
